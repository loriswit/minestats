minestats/server

API

GET /status get the server status
GET /logs get the latest logs
GET /archived get a list of archived servers
GET /users
GET /archived/{server}/users 		get a list of all players
GET /stats/{uuid}
GET /archived/{server}/stats/{uuid} 		get the statistics of a specific player
GET /advancements/{uuid}
GET /archived/{server}/advancements/{uuid} 		get the advancements of a specific player
